Основная суть проекта создание футбольного сайта.
Проект создан при помощи WEB.
Код состоит из множества обработчиков, создающих страницы.
Проект содержит множество html файлов с разным содержимым.
Большая часть html файлов содержат информацию о клубах и визуальную составляющую их страниц.
5 файлов отвечают за создание подробной трнирной таблицы по каждому чемпионату.
Один файл отвчает засоздание стартовой страницы со списком чемпионатов.
Папка img содержит все задние фона страниц.
